# 目标大学 Demo

## 打开方式

直接打开：

```text
demo/index.html
```

或启动静态服务：

```bash
python3 -m http.server 8787 --bind 127.0.0.1
```

访问：

```text
http://127.0.0.1:8787/demo/index.html
```

## 可直接访问的页面

- `#profile` 目标档案
- `#schools` 选择目标大学
- `#goals` 成绩填报

## 覆盖流程

目标档案 -> 选目标大学 -> 搜索/筛选 -> 设置目标大学 -> 确认选择 -> 签署目标大学军令状 -> 返回目标档案 -> 左侧维护目标成绩并选择成绩单 -> 右侧填报考试成绩。
